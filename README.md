## Credit Card Fraud Detection

The **Credit Card Fraud Detection** notebook focuses on detecting fraudulent credit card transactions using the AdaBoost, XGBoost, and RandomForestClassifier algorithms. The notebook demonstrates data preprocessing, model training, and evaluation techniques specific to credit card fraud detection tasks.

## Fetal Health Classifier

The **Fetal Health Classifier** notebook aims to classify the health status of fetuses based on various physiological parameters. It utilizes the AdaBoost, XGBoost, and RandomForestClassifier algorithms for classification. The notebook provides insights into feature engineering, model training, and performance evaluation for fetal health classification.

## Recommendation System Collaborative Filtering

The **Recommendation System Collaborative Filtering** notebook implements a recommendation system using the K-Nearest Neighbors (KNN) algorithm. The notebook focuses on collaborative filtering techniques to provide personalized recommendations based on user-item interactions. It demonstrates how to preprocess data, train the KNN model, and generate recommendations.

## Gemstone Price Prediction

The **Gemstone Price Prediction** notebook addresses the task of predicting the price of gemstones using MLP regression and Gradient Boosting Regression algorithms. The notebook covers data preprocessing, model training, hyperparameter tuning, and evaluation techniques specific to gemstone price prediction.
